What is Typescript?

TypeScript is a superset of JavaScript that includes static types, which is a feature that JavaScript does not have. This means that variables, function parameters, and return values can have explicit types like in Java or C++. TypeScript allows the addition of generics and Object Oriented Programming (OOP) features to JavaScript, including classes, inheritence, and interfaces. However, TypeScript code has to be compiled into JavaScript using the TypeScript compiler (tsc) before it can be used in HTML files to create web pages.

What are the advantages of TypeScript?

TypeScript allowing the option of static typing to JavaScript is helpful because it can more efficiently catch errors. This is because static typing allows type errors to be caught during compilation instead of during runtime, which improves the development process. Additionally, I personally appreciate the static typing because it makes the code more readable; at first glance, I gain more information about what each variable or function does because its type information is written right there. However, without this feature, it would be more difficult to understand JavaScript code and what its purpose is.
